## 🚀 Key Features

- 🧠 **AI Formula Generator** — Describe your logic in plain English, and AiXL compiles it instantly into syntactically correct Excel formulas (including nested lookups, index-matches, and array functions).
- 🧹 **Automated Data Cleaning** — Standardize raw names, format phone numbers, parse addresses, and trim whitespaces in a click without custom VBA scripts.
- 📊 **Smart Trend Analysis** — Query your dataset using natural language to extract professional insights, summarize rows, and preview visual trends.
- 🔒 **Security & Privacy First** — Connect via secure local endpoints or enterprise API keys. Spreadsheets remain confidential and are never used for public training.

## 📦 Getting Started & Installation

AiXL operates as a lightweight manifest-based developer add-in. Follow these steps to register:

1. **Download the Manifest** — Get the latest addon release package folder from [GitHub releases](https://github.com/kaileyventures).
2. **Open Add-ins Manager** — Launch Microsoft Excel, head to the top ribbon and select **Insert** > **My Add-ins** > **Developer Add-ins**.
3. **Register Manifest XML** — Locate the downloaded manifest XML configuration file and click register to initialize the side panel.

*Supports Excel 2019, 2021, and Excel 365 on Windows, macOS, and Web.*
